# История изменений

Все заметные изменения проекта фиксируются здесь.

Формат близок к [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/), версии
ведутся по смыслу: `major.minor.patch`.

## 1.5.1 — 12-06-2026

### Добавлено

- Пункт меню `Сменить аккаунт`: он закрывает текущий браузерный контекст,
  удаляет локальный профиль оверлея и открывает чистое окно входа NeuroGate.

### Изменено

- Оверлей больше не нажимает кнопку входа автоматически, даже если форма уже
  заполнена сохранёнными данными браузера. Пользователь сам подтверждает вход
  на сайте NeuroGate.
- README и `docs/PRIVACY.md` уточняют, что cookies и сессия хранятся только
  локально в `%USERPROFILE%\.neurogate-usage-overlay\browser-profile` и не
  попадают в GitHub/ZIP-релиз.

### Проверки

- Проверки проекта: `51 tests OK`.

## 1.5.0 — 10-06-2026

### Добавлено

- Проверка новой версии при запуске и затем раз в сутки через публичные GitHub Releases.
- Пункт меню `Обновить до vX.Y.Z`, который появляется только при доступном обновлении.
- Скрипт `scripts/update-and-restart.ps1` для безопасного обновления и перезапуска.
- Корневой установщик `Install-NeuroGate-API.bat` для установки из ZIP без Codex,
  Claude Code и Git.
- Скрипт `scripts/package-release.ps1` для сборки ZIP-архива под GitHub Releases.

### Изменено

- Обновление программы отделено от обновления лимитов: пользователь сам запускает
  установку новой версии из меню.
- Git-установка обновляется через `git pull --ff-only`, а ZIP-установка скачивает
  ZIP новой версии с GitHub и раскладывает его поверх текущей папки.
- Автообновление Git-установки останавливается при локальных изменениях в папке
  проекта, чтобы не перезаписать пользовательские правки.

## 1.4.0 — 09-06-2026

### Добавлено

- Галочка `2x размер` в меню по правой кнопке.
- Пропорциональное увеличение оверлея в два раза: окно, координаты, шрифты,
  линии, прогресс-бары и подсказки масштабируются вместе.
- Сохранение выбранного размера в локальном `overlay-state.json`.

### Проверки

- Добавлены тесты на сохранение масштаба и ограничение позиции окна с учётом
  увеличенного размера.

## 1.3.0 — 08-06-2026

### Добавлено

- Подсказки при наведении на оранжевые цифры лимитов.
- Для `5ч` показывается `Потрачено со сброса`.
- Для `7д` показывается `сегодня потрачено` или `сегодня потрачено с HH:MM`, чтобы пользователь видел точку
  отсчёта дневного расхода.
- Локальный файл `usage-daily.json` для расчёта дневного расхода по `7д`.
- Автонажатие кнопки входа, если видимая форма уже заполнена сохранёнными данными браузера.
  Позже это поведение удалено в `1.5.1`.

### Изменено

- Дневная точка отсчёта хранит только текущий день и перезаписывается при смене
  даты, поэтому файл не разрастается при длительном использовании.

## 1.2.1 — 08-06-2026

### Исправлено

- Исправлено сопоставление полосок прогресса после сброса лимитов.
- Если у карточки лимита на сайте нет синей заливки, оверлей теперь считает это
  `0%`, а не пропускает карточку.
- Проценты прогресса теперь привязываются к окнам `5ч`, `24ч`, `7д` по названию,
  а не по порядку элементов на странице.

### Проверки

- Добавлены регрессионные тесты на случай, когда первая полоска отсутствует.
- Проверки проекта: `27 tests OK`.

## 1.2.0 — 07-06-2026

### Проверки

  пользовательских настроек.
- Проверки проекта: `25 tests OK`.

## 1.1.0 — 07-06-2026

### Добавлено

- Полоски прогресса лимитов, совпадающие с полосками на сайте NeuroGate.
- Динамическое чтение названия тарифа со страницы вместо жёсткой привязки к

- Один актуальный скриншот в README вместо двух старых.

- Старый файл `docs/screenshots/overlay-menu.png` удалён.

### Исправлено

- При нулевом прогрессе синяя полоска больше не рисует искусственный хвостик.

### Проверки

- Добавлены тесты на динамическое имя тарифа и проценты прогресса.
- Проверки проекта: `22 tests OK`.

## 1.0.0 — 07-06-2026

### Добавлено

- Компактный always-on-top оверлей для Windows.
- Чтение страницы лимитов NeuroGate через локальный профиль Chrome.
- Показ тарифа, времени действия тарифа, остатков `5ч` и `7д`, времени сброса и
  последнего обновления.
- Компактное меню по правой кнопке.
- Пункт `Не закрывать ЛК`.
- Скрытие окна Chrome после успешного входа.
- Быстрое обновление состояния входа, если сайт требует авторизацию.
- Ярлык на рабочем столе через установочный скрипт.
- Русскоязычная пользовательская инструкция по установке через Codex/Claude.

### Проверки

- Базовые тесты парсера, браузерного режима и UI-поведения.
